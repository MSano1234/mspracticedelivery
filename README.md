# рҹ“– Overview

MS Cloud Delivery Platform is a cloud-native web application designed to demonstrate practical AWS engineering skills using a fully serverless architecture.

The application allows users to create, manage, update, and monitor delivery requests through a responsive React application while leveraging AWS managed services for scalability, reliability, and low operational overhead.

This project showcases real-world cloud engineering practices including:

- Serverless Computing
- REST API Development
- Infrastructure as Code
- Cloud Deployment
- NoSQL Database Design
- Modern Frontend Development

